Filosofía Inversor Directivo

  • Margen de seguridad ≥25%: el alquiler puede caer un 25% y seguir generando cashflow positivo.
  • Zonas C en escala ABCD: mejor binomio rentabilidad/riesgo. Barrios obreros estables.
  • Pisos antiguos (>15 años): las casas nuevas no rentan. Pero atención a derramas en >30 años.
  • Evitar plantas bajas: las preferidas de los okupas (probabilidad real 0,14% pero medida prudente).
  • Vacancia mín 8% · mantenimiento 10%: cálculos prudentes, no optimistas.
  • Rentabilidad bruta >6%: por debajo, el cashflow es precario. Umbral de descarte rápido.
  • Reformas integrales: pagar máximo el 50% del valor post-reforma. Si no, no compensa el dolor de cabeza.
  • Probabilidad Nike: los chollos están al inicio (precio mal puesto) o tras meses sin moverse (vendedor cansado).
  • Preferir <150k€: rango con mayor concentración de oportunidades.

Conexión con GitHub

El token se guarda solo en tu navegador (localStorage). Nunca se envía a ningún servidor excepto a la API oficial de GitHub.

Cómo generar el token

  1. Ve a github.com/settings/tokens (Fine-grained)
  2. Click en Generate new token
  3. Token name: lo que quieras (ej: dashboard-inmobiliario)
  4. Expiration: máximo (1 año)
  5. Repository access: Only select repositories → tu repo concreto
  6. Permissions → Repository permissions → Contents: Read and write
  7. Generate token → cópialo y pégalo arriba (no lo verás dos veces)

Acciones

Elimina el token y la configuración de este navegador. Tendrás que volver a configurarlo.
